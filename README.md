# Parayaz

Turkish number to text money

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'parayaz'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install parayaz

## Usage
```ruby
> 123.parayaz
# yüzyirmiüçTL

> 123.55.parayaz
# yüzyirmiüçTL,ellibeşkr
```
